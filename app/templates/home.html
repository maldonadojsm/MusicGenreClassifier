Hello there! Welcome to Scriabin, a music genre classifier designed using a Convolutional Neural Network (CNN). This web application intends to classify audio music based on ten types of genres (Blues, Classical, Country, Disco, Hip Hop, Jazz, Metal, Reggae, Rock). The model performing inferences currently exceeds human accuracy by 12%.

Please consult the following link to discover how the model was trained: (https://github.com/maldonadojsm/Scriabin).

Dataset Used: (http://marsyas.info/downloads/datasets.html
